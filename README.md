> **"가설 검증부터 배포까지 주도하는 Product Engineer Baeju0의 작업실"**

이 저장소는 저의 개발 가치관, 운영 중인 제품(Products), 그리고 기술적 경험을 담은 포트폴리오 웹사이트의 소스 코드입니다. **Next.js 15**와 **Tailwind CSS**를 활용하여 구축되었으며, 미니멀한 디자인과 반응형 UI, 그리고 검색 엔진 최적화(SEO)에 집중했습니다.

## 🧩 Core Values

저는 단순한 기능 구현을 넘어, 비즈니스 임팩트를 만드는 엔지니어링을 지향합니다.

1. **기술보다 비즈니스 임팩트** 화려한 최신 기술보다, 문제를 가장 확실하고 효율적으로 해결하는 '적정 기술'을 지향합니다.
2. **AI를 도구가 아닌 동료로** 단순 코딩 보조를 넘어, 기획과 설계를 함께 논의하는 파트너(Agent)로 활용합니다.
3. **만드는 편함보다 쓰는 경험** 개발자의 편의성보다 사용자의 1초를 아끼는 UX를 최우선으로 선택합니다.
4. **완벽한 준비보다 빠른 실행** 거대한 계획보다 작은 실험을 신뢰하며, 빠르게 검증하고 데이터로 배웁니다.
